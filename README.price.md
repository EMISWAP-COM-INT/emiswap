# Pricedump contract README

Контракт предназначен для получения информации о ценах токенов на биржах типа Uniswap одним массивом.

## Методы контракта

### getCoinPrices
Метод принимает на вход список адресов токенов, для которых необходимо получить цену и индекс-указатель на биржу, с которой необходимо получать эти цены.
Метод возвращает массив цен указанных токенов по отношению к DAI, умноженных на 100000 (10 в степени 5) для исключения ошибок округления.
Т.е. для получения точной цены с точностью до 5 знака после запятой необходимо полученное значение привести к формату с плавающей точкой и поделить на 100000.
Если цену для какого-то токена определить не удалось, в соответствующей позиции возвращается 0.

Значения параметра _market:
 - 0: наша собственная биржа
 - 1: биржа Uniswap
 - 2: биржа Mooniswap

```solidity
  function getCoinPrices(address [] calldata _coins, uint8 _market) external view returns(uint[] memory prices)
```
